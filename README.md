# 20234039 컴퓨터공학과 장태희 진로계획

# -단기목표-
-전공 기초 공부: 자료구조, 알고리즘, 머신러닝 등 과의 뼈대가 되는 전공 깊이있게 학습하기.
-관심 분야 선정: 컴퓨터공학에서 배우는 다양한 분야중 내가 관심있는것을 선정하여 따로 깊게 공부해보기.

# -중기목표-
-교외 프로젝트 참여: 교외에서 진행하는 프로젝트나 대회를 참가하여 수상을 목표로 해보기.
-깃허브에 내 프로젝트들 공유: 나중에 중요한 포트폴리오 자료가 될 수도 있기에 틈 날때 한 작업물을 깃허브에 올려 저장하기.
-취업 준비하기: 방학동안,아니면 4학년 2학기 전까지 정보처리기사, 컴퓨터활용능력 1급 등을 준비하여 스펙 마련하기.

#  -장기목표-
-개인 연구: 취업 후 회사에서 진행하는 일 이외에도 내가 관심있는 분야 꾸준히 공부해보기.
-트렌드 파악: 최신 트렌드를 파악하여 시대에 맞는 개발자가 무엇일까를 계속 고민해보기.
